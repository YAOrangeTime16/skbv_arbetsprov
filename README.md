# Salong-App
     
This application has been created with React.js and `create-react-app`.
     
### Installation
```sh
  git clone https://github.com/YAOrangeTime16/salong-app.git
  cd salong-app
```
then
     
```sh
  npm install
  npm start
```
or 
   
```sh
  yarn
  yarn start
```
