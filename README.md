# skbv_arbetsprov
     
This application has been created with React.js and `create-react-app`.
The app has been deployed at https://skbv-a03b9.firebaseapp.com/
     
### Installation
```sh
  git clone https://github.com/YAOrangeTime16/skbv_arbetsprov.git
  cd skbv_arbetsprov
```
then
     
```sh
  npm install
  npm start
```
or 
   
```sh
  yarn
  yarn start
```
